## step1: 准备工作
在新的 ubuntu 22.04 更新、升级、安装必要软件

1）系统软件更新的升级
```
sudo apt update
sudo apt upgrade
```
上述指令没有VPN的或者其它原因可能会失败，可以安装VPN后再尝试运行

2）安装vpn 

3）换源
```
wget http://fishros.com/install -O fishros && . fishros
```
4） 安装中文输入法，取决于自己喜好，这里安装google pinyin
https://cloud.tencent.com/developer/article/2417968

5)  安装vscode
```
wget http://fishros.com/install -O fishros && . fishros
```
